EPT Vision

An AI-powered extension of the Esperando Por Ti (EPT) platform, revolutionizing animal welfare in Yucatán through advanced computer vision. Our system helps shelters and rescuers by detecting health issues, assessing nutrition status, identifying pregnancies, and matching lost pets with their families.

"The vision that protects lives"

System Architecture

High-level system architecture showing the interaction between frontend, ML pipeline, and infrastructure components.

Detailed model architecture showing the multi-task learning approach with shared backbone and task-specific heads.

ML Models & Training

Base Model: EfficientNet B0 (pretrained)
Architecture: Multi-task learning with shared backbone
Hyperparameters:
Batch Size: 32
Learning Rate: 0.001
Epochs: 50
Weight Decay: 1e-4
Task Weights:
Animal Type: 1.0
Health Issues: 1.2
Size: 0.8
Pregnancy: 0.9
Quality: 0.7
Training Metrics:
Animal Type Accuracy: 93.94%
Body Condition Accuracy: 88.79%
Context Accuracy: 94.62%
Health Issues Accuracy: 91.02%
Training Details:
Dataset: 1000+ labeled images from local shelters
Augmentation: Rotation, flips, color jittering
Optimizer: AdamW with learning rate scheduling
Loss: Multi-task weighted cross-entropy

Dataset Creation & Pre-labeling

Automated Pre-labeling Pipeline

Our pre-labeling tool significantly reduces manual labeling effort by:

  • Initial Filtering:
    • Using YOLOv8x-seg for animal detection
    • Automatic discard of non-animal images
    • Quality assessment filtering
  • Multi-Model Approach:
    • EfficientNet B0 backbone for feature extraction
    • Specialized disease detection model
    • Dedicated pregnancy detection model
    • Nutrition analysis model
  • Efficiency Metrics:
    • ~80% reduction in manual labeling time
    • 94% accuracy in animal detection
    • Real-time processing capabilities
